Aprendizados

Hoje, aprendi sobre autenticação com o Django Rest Framework e adicionei um endpoint de autenticação de token à API usando a visão obtain_auth_token.

Estou usando HTTPie e para gerar um token, você só precisa fazer um post no endpoint, no meu caso auth-token/, com o nome de usuário e senha:

http POST :8000/auth-token/ password='password' username='username'

Ainda estou tendo alguns problemas para entender como realmente usar o token como autenticador, mas, de acordo com a documentação do httpie, deve se parecer com algo assim:

http -A bearer -a 448b01551721565eaa5f0aff2e33c206ce571c5a :8000/snippets/ code="print(123)"

Também encontrei alguns sites (como este) dizendo que eu deveria usar algo assim:

http :8000/endpoint/ 'Authorization: Token 9054f7aa9305e012b3c2300408c3dfdf390fcddf'

mas não consegui fazê-lo funcionar para POST requests.

Para autenticar com nome de usuário e senha (autenticação básica), usamos isso:

http -a username:password :8000/snippets/ code="print(123)"
