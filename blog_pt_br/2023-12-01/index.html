Aprendizados

Hoje aprendi que, para fazer o mockito funcionar, tive que usar a anotação @GenerateNiceMocks para informar ao mockito quais classes mockar e o build runner do Dart para gerar um arquivo .mock.dart. No final, o código parece mais ou menos assim:

@GenerateNiceMocks([MockSpec<ClassName>()])
import 'file_name_test.mocks.dart';

e depois de adicionar isso, você precisa executar:

dart run build_runner build

para gerar o arquivo file_name_test.mocks.dart.
