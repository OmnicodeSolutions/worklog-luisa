Learnings

Today I learned that to make mockito work I had to use the @GenerateNiceMocks annotation to tell mockito which classes to mock and Dart's build runner to generate a .mock.dart file. In the end, the code looks kinda like this:

@GenerateNiceMocks([MockSpec<ClassName>()])
import 'file_name_test.mocks.dart';

and after adding that you have to run:

dart run build_runner build

to generate file_name_test.mocks.dart.
